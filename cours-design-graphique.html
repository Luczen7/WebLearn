1 Introduction au Design Graphique

Le design graphique est l'art de communiquer visuellement. Il combine texte, images et éléments graphiques pour transmettre un message de manière claire, esthétique et efficace.

1.1 Les 10 principes fondamentaux du design

  • Équilibre : Distribution visuelle des éléments (symétrique, asymétrique, radial)
  • Contraste : Différenciation par la couleur, taille, forme, texture
  • Hiérarchie : Organisation de l'importance visuelle
  • Répétition : Cohérence et unité visuelle
  • Alignement : Création de liens visuels entre les éléments
  • Proximité : Regroupement des éléments liés
  • Espace blanc (Negative Space) : L'espace vide est aussi important que le plein
  • Mouvement : Direction du regard du spectateur
  • Rythme : Répétition régulière créant un flux visuel
  • Unité : Harmonie entre tous les éléments

1.2 Processus de design

  1. Brief : Comprendre les besoins et objectifs
  2. Recherche : Analyser le marché, la concurrence, le public cible
  3. Conceptualisation : Brainstorming et esquisses
  4. Création : Développement des maquettes
  5. Itération : Feedback et améliorations
  6. Finalisation : Livraison des fichiers finaux

2 Théorie des Couleurs

2.1 Le cercle chromatique

Le cercle chromatique est l'outil fondamental pour comprendre les relations entre les couleurs :

  • Couleurs primaires : Rouge, Jaune, Bleu (RYB) ou Rouge, Vert, Bleu (RVB)
  • Couleurs secondaires : Orange, Vert, Violet (mélange de deux primaires)
  • Couleurs tertiaires : Jaune-orange, Rouge-orange, Rouge-violet, etc.

2.2 Harmonies de couleurs

HarmonieDescriptionUsage
ComplémentaireDeux couleurs opposées sur le cercleContraste fort, CTA
TriadiqueTrois couleurs équidistantesVibrant, équilibré
AnalogueCouleurs adjacentesHarmonieux, naturel
Complémentaire partagéeUne couleur + deux adjacentes à son opposéContraste nuancé
MonochromatiqueVariations de teinte, saturation, luminositéÉlégant, minimaliste

2.3 Psychologie des couleurs

  • Rouge : Énergie, passion, urgence, danger
  • Bleu : Confiance, calme, professionnalisme, technologie
  • Vert : Nature, croissance, santé, argent
  • Jaune : Optimisme, créativité, attention
  • Orange : Enthousiasme, chaleur, action
  • Violet : Luxe, créativité, spiritualité
  • Noir : Élégance, pouvoir, mystère
  • Blanc : Pureté, simplicité, espace

2.4 Systèmes de couleurs numériques

Design - Systèmes Couleur 
// Formats de couleur en design digital :

// HEX (Hexadécimal)
#FF5733 = Rouge-orangé
#1A3A6E = Bleu foncé (WebLearn primary)
#F26522 = Orange (WebLearn secondary)

// RGB (Red, Green, Blue)
rgb(255, 87, 51)
rgba(255, 87, 51, 0.8)  // avec opacité 80%

// HSL (Hue, Saturation, Lightness)
hsl(9, 100%, 60%)
// Hue : 0-360° (position sur le cercle)
// Saturation : 0-100% (intensité)
// Lightness : 0-100% (luminosité)

// HSB/HSV (Hue, Saturation, Brightness/Value)
// Similaire à HSL mais Brightness au lieu de Lightness

// Outils de palette :
- Coolors.co (générateur de palettes)
- Adobe Color (harmonies basées sur le cercle)
- ColorHunt (palettes communautaires)
- Paletton (harmonies avancées)
Règle des 60-30-10

Pour une palette harmonieuse, utilisez : 60% de couleur dominante (neutre), 30% de couleur secondaire, et 10% d'accent (couleur vive pour attirer l'attention).

3 Typographie

La typographie est l'art d'organiser le texte dans un design. Un bon choix typographique peut transformer un design moyen en design exceptionnel.

3.1 Classification des polices

CatégorieCaractéristiquesExemplesUsage
SerifEmpattements, traditionnelTimes, Georgia, PlayfairÉditorial, luxe
Sans-serifSans empattements, moderneHelvetica, Arial, InterUI, tech, web
ScriptStyle manuscritPacifico, Dancing ScriptInvitations, branding
DisplayDécorative, impactanteImpact, Bebas NeueTitres, posters
MonospaceLargeur fixeCourier, Fira CodeCode, data

3.2 Anatomie typographique

  • Baseline : Ligne de base sur laquelle reposent les lettres
  • X-height : Hauteur des lettres minuscules (a, e, x)
  • Ascender : Partie au-dessus de la x-height (b, d, h)
  • Descender : Partie en dessous de la baseline (g, j, p)
  • Kerning : Espacement entre deux lettres spécifiques
  • Tracking (letter-spacing) : Espacement uniforme entre toutes les lettres
  • Leading (line-height) : Espacement entre les lignes

3.3 Hiérarchie typographique

Design - Échelle Typographique 
// Échelle typographique modulaire (ratio 1.25 - Major Third)

Display Large  : 96px  / line-height: 1.1  / weight: 700
Display Medium : 60px  / line-height: 1.1  / weight: 700
H1             : 48px  / line-height: 1.2  / weight: 700
H2             : 40px  / line-height: 1.2  / weight: 600
H3             : 33px  / line-height: 1.3  / weight: 600
H4             : 28px  / line-height: 1.3  / weight: 600
H5             : 23px  / line-height: 1.4  / weight: 500
H6             : 19px  / line-height: 1.4  / weight: 500
Body Large     : 18px  / line-height: 1.6  / weight: 400
Body           : 16px  / line-height: 1.6  / weight: 400
Body Small     : 14px  / line-height: 1.5  / weight: 400
Caption        : 12px  / line-height: 1.4  / weight: 400

// Règles d'or :
// - line-height corps de texte : 1.5 - 1.7
// - line-height titres : 1.1 - 1.3
// - Longueur de ligne idéale : 45-75 caractères
// - Contraste minimum : 4.5:1 (WCAG AA)

3.4 Pairing de polices

Associer deux polices complémentaires :

  • Contraste de style : Serif + Sans-serif (Playfair + Inter)
  • Contraste de poids : Light + Bold
  • Même famille : Utiliser les variations d'une super-famille
  • Maximum 2-3 polices par projet

Combinaisons populaires :

  • Merriweather + Open Sans (blog, éditorial)
  • Montserrat + Lato (corporate, startup)
  • Poppins + Roboto (tech, moderne)
  • Playfair Display + Source Sans Pro (luxe, élégant)

4 Composition et Grid Systems

4.1 Règles de composition

  • Règle des tiers : Diviser l'image en 9 parties égales, placer les points d'intérêt aux intersections
  • Point focal : Un seul élément dominant qui attire l'attention
  • Lignes directrices : Utiliser les lignes naturelles pour guider le regard
  • Cadre et bordure : Encadrer le sujet pour le mettre en valeur
  • Symétrie et asymétrie : Créer de l'équilibre ou du dynamisme

4.2 Systèmes de grille

Design - Grid Systems 
// Grille de colonnes (système le plus courant)

// Desktop (1440px)
Columns  : 12
Gutter   : 24px
Margin   : 48px
Column width : (1440 - 96 - 264) / 12 = 90px

// Tablet (768px)
Columns  : 8
Gutter   : 24px
Margin   : 32px

// Mobile (375px)
Columns  : 4
Gutter   : 16px
Margin   : 16px

// Types de grilles :
// 1. Manuscript grid : Une seule colonne (livres)
// 2. Column grid : Plusieurs colonnes égales (web, magazines)
// 3. Modular grid : Colonnes + lignes (tableaux de données)
// 4. Hierarchical grid : Grille irrégulière (posters créatifs)
// 5. Baseline grid : Alignement sur une grille de lignes (typographie)

4.3 Golden Ratio (Ratio d'or)

Le nombre d'or (φ ≈ 1.618) est considéré comme esthétiquement plaisant :

  • Proportion des éléments : 1 : 1.618
  • Spirale d'or pour la composition
  • Échelle typographique basée sur φ
  • Exemple : 16px → 26px → 42px → 68px

5 Identité Visuelle et Branding

5.1 Éléments d'une identité visuelle

  • Logo : Symbole principal de la marque
  • Palette de couleurs : Couleurs primaires, secondaires, neutres
  • Typographie : Polices de marque pour tous les supports
  • Imagery : Style photographique, illustrations
  • Voice & Tone : Personnalité de la communication
  • Patterns & Textures : Éléments décoratifs récurrents

5.2 Création d'un logo

Types de logos :

  • Wordmark : Texte stylisé (Google, Coca-Cola)
  • Lettermark : Initiales (IBM, HBO, NASA)
  • Pictorial mark : Symbole iconique (Apple, Twitter)
  • Abstract mark : Forme abstraite (Nike, Pepsi)
  • Mascot : Personnage (KFC, Michelin)
  • Combination mark : Texte + symbole (Adidas, Lacoste)
  • Emblem : Texte dans un symbole (Harvard, Starbucks)

5.3 Brand guidelines

Un guide de marque (brand book) contient :

  • Positionnement et valeurs de la marque
  • Usage correct et incorrect du logo
  • Palette de couleurs avec codes exacts
  • Polices et hiérarchie typographique
  • Templates pour différents supports
  • Photographie et style visuel
  • Tone of voice

Projet final : Identité de marque complète

Créez une identité visuelle complète pour une marque fictive (café, application, ou boutique) incluant : recherche et moodboard, logo avec variations, palette de couleurs avec codes, typographie de marque, cartes de visite mockup, mockup de packaging ou d'application, et un mini brand book de 4-6 pages.
